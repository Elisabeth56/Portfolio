# Images Folder

Place your images here. Required images:

## Hero Background
- **Filename:** `hero.png`
- **Recommended size:** 1920x1080 or larger
- **Usage:** Full-screen background for the hero section
- **Note:** The site works without this image (shows gradient fallback)

## Profile Image (Optional)
- **Filename:** `profile.png`
- **Recommended size:** 800x1000
- **Usage:** About Me section

## Tips
- Use high-quality images (WebP or optimized JPG)
- Hero image should work well with dark overlays
- Images are automatically optimized by Next.js
- The purple/blue/cyan gradient will show as fallback if images are missing
